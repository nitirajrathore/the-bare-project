### Install

```
npm init -y
```

```
npm install webpack webpack-cli --save-dev
```

#### later on
```
npm install --save lodash
```

#### run webpack with config
```
npx webpack --config webpack.config.js
```

#### run with npm
```
npm run build
```
#### handle dynamic html updation
```
npm install --save-dev html-webpack-plugin
```
